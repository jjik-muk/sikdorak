# 서비스 소개 📝

사용자(나, 친구)는 특정 음식점의 솔직한 리뷰를 작성합니다!  

작성한 리뷰는 친구 관계(팔로잉/팔로워)에만 공개됩니다.  

솔직한 리뷰를 작성하는 미식가(친구)를 팔로잉하고 식도락을 떠나보세요!
